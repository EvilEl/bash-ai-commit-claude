## Автоматическая генерация сообщений коммитов с помощью Claude

Если вы хотите, чтобы Claude автоматически описывал ваши изменения в коммит-сообщениях, воспользуйтесь скриптом `git-commit`.

### Установка

1. Скопируйте файл `git-commit` в директорию `~/bin/`:
```bash
cp git-commit.sh ~/bin/
chmod +x ~/bin/git-commit
```

2. Убедитесь, что `~/bin` находится в вашем PATH (обычно уже есть в Git Bash).

### Использование
```bash
# Добавьте изменения в staging
git add .

# Запустите скрипт
git-commit
```

Скрипт проанализирует ваши изменения и предложит сообщение коммита в формате conventional commits. Вы сможете подтвердить или отклонить предложенное сообщение.

### Для удобства

Создайте alias, добавив в `~/.bashrc`:
```bash
alias gac="bash ~/bin/git-commit"
```

Теперь можно использовать просто:
```bash
git add .
gac
```

### Требования

- Установленный [Claude CLI](https://github.com/anthropics/claude-cli)
- Git
